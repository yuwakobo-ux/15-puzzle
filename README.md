# Mini Slide Puzzle Ver0.1

A very small static 3x3 sliding puzzle app for smartphone-friendly browser play.

## Files

- `index.html` - app page
- `style.css` - layout and visual style
- `script.js` - puzzle logic
- `OPEN_ME_FIRST.html` - simple launcher page

## How to open

Open `index.html` in a browser. No server, login, upload, database, API, or package install is required.

## How to test

1. Open `index.html`.
2. Tap `Shuffle`.
3. Tap a numbered tile next to the empty space.
4. Confirm the move count increases.
5. Tap `Reset` to return to the solved board.
6. Solve the puzzle and confirm the `Clear!` message appears.

## Known limitations

- Ver0.1 uses numbers only, not images.
- There is no timer, best score, animation between grid cells, or saved progress.
- Keyboard controls are not included.
